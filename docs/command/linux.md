```shell
# 创建目录
mkdir 目录名称

# 查看当前目录路径
pwd

# 查看当前文件和文件夹
ls

# 查看当前文件和文件夹包括隐藏文件
ls -al

# 修改目录的名称
mv 目录名称 新目录名称

# 剪切
mv 目录名称 新目录地址

# 拷贝目录 (-r代表递归拷贝)
cp -r 目录名称 目录拷贝的目标位置

```

```shell
# 文件的创建
touch 文件名称

# 文件的查看
cat  文件名称

# 删除文件
rm -rf 文件名

# 压缩文件 (z：调用gzip压缩命令进行压缩  c：打包文件  v：显示运行过程   f：指定文件名)
tar -zcvf 打包压缩后的文件名 要打包压缩的文件

# 解压文件
tar [-xvf] 压缩文件  解压地址

```

```shell
# 搜索命令  要搜索的字符串 要搜索的文件
grep

# 管道命令 (把上一次的结果当做下一次的参数) 例如（grep 匹配内容，一般结合管道符使用）文件或目录为Git的 -> ls | grep Git
|

#查看当前系统中运行的进程 
ps -ef  

# 杀死进程
kill -9 进程的pid

# 查看当前系统的网卡信息
ifconfig

# 查看操作历史
history

# SSH登录
root@192.168.0.1

# 重定向 (> 覆盖    >> 追加)
echo hello world >> README.md     

# 网络请求 (例如:获取网站的html文本(curl http://www.baidu.com >> baidu.html))
curl http://www.baidu.com  

# 查看当前用户
who am i

# 查看路径 (查看git路径)
which git

```

```shell
# VI 编辑器
# 命令行模式命令
*  ZZ（大写）保存并退出
*  u辙销操作，可多次使用
*  dd删除当前行
*  yy复制当前行
*  p 粘贴内容
*  ctrl+f向前翻页
*  ctrl+b向后翻页
*  i进入编辑模式，当前光标处插入
*  a进入编辑模式，当前光标后插入
*  A进入编辑模式，光标移动到行尾
*  o进入编辑模式，当前行下面插入新行
*  O进入编辑模式，当前行上面插入新行



# 底行模式命令
* :w保存，:w filenme另存为
* :q退出
* :wq保存并退出
* :e! 撤销更改，返回到上一次保存的状态
* :q! 不保存强制退出
* :set nu 设置行号


```
